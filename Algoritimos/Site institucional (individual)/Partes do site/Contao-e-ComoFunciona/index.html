Como funciona

O orgão é transportado por helicopteros, aviões ou ambulâncias, dentro de uma caixa isotérmica, equipada com um sensor de temperatura capaz de transmitir os dados de temperatura para a nuvem, onde serão gerados para o usuário e por fim o registro de todo o transporte.
